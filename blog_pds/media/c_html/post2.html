Instantiating a Class

Here we are going to instante a class in a main method

Right click on the directory "Propeties" and then Add->New File...->General->Empty Class->New

So there we created a Class "Dog" that contains a method called "bark". When it is called, the console gets a "Roof!" printed

using System; namespace Tutorial { //Created a public class named Dog public class Dog { //Method from Dog class called bark that prints "Roof!" at the console public void bark(){ Console.WriteLine ("Roof!"); } } }

Now we are creating an instantiation of the Class Dog and then using its method to print at the console.

using System; namespace Tutorial { class MainClass { public static void Main (string[] args) { //Creating an instantiation of the Dog Class Dog Coco = new Dog (); //Using the Dog method called "bark" to print "Roof!" a the console Coco.bark (); } } }
