Creating the first class and printing a hello world!

First, lets open the MonoDevelop. We are using MonoDevelop software because it is free and it is more likely C# sintax than VisualStudio sintax. C# sintax is simmilar to Java (for those familiar).

Click on "New", and then "Console Project" (Remember of selecting C# at the right).

Then choose the name of your project at the Project Name field and Also the directory you want to put.

Now we got a page called "Program.cs". If dont, try right click on the directory "Propeties" and then Add->New File...->General->Empty Class->New

So we got a class declaration named "MainClass" inside a "namespace" statement called "Tutorial".

//The namespace keyword is used to declare a scope that contains a set of related objects. You can use a namespace to organize code //elements and to create globally unique types. See more at: https://msdn.microsoft.com/en-us/library/z2kcy19k.aspx namespace Tutorial { //Declaration of a class class MainClass { } }

Now we got a sample code with a file named Tutorial.cs that prints a "Hello World!" at the console.

// This prevents the user to no use the System statement at some points // for example: System.Console.WriteLine ("Hello World!"); using System; namespace Tutorial { class MainClass { //This is the main method of the program public static void Main (string[] args) { //This prints a "Hello World!" at the console Console.WriteLine ("Hello World!"); } } }

The result is a "Hello World!" at the console as shown. Press F5 to Run.
