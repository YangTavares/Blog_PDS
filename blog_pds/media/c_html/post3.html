Class properties and variables

Here we are goint to know how to create,expose and access properties of the classes.

First we need to say if the variable is "public" or "private", the public statement means that you can access the variable out of its class and the private you cannot access the variable out of the class.

There are two ways to access the public variables at a class, you can create methods and access it directly as in Java, or you can create the get-set accessors.

The get-set accessors uses a method made as shown. "get" is used to return a value when the "Size" is called and "set" is used to assign a "value" to the variable. The "value" statement gets the assigned value passed by the " = " operator.
